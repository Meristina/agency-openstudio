# Budget Actuals — méthode

## Objectif
Appliquer la méthode **budget-actuals** pour produire un livrable actionnable, sourcé, calibré sur le contexte.

## Procédure

1. **Cadrage** — Comprendre le contexte (secteur, taille, stade, contraintes)
2. **Collecte** — Rassembler les données disponibles, identifier les lacunes
3. **Application** — Appliquer la méthode avec rigueur : aucun chiffre inventé
4. **Validation** — Vérifier la cohérence interne et les hypothèses
5. **Livraison** — Produire le livrable formaté pour un décideur

## Standards de qualité
- Aucun chiffre non sourcé (benchmark cité ou hypothèse labelisée)
- Recommandations actionnables avec ownership suggéré
- Limites et incertitudes documentées
- Cohérence avec les autres outputs de la mission
